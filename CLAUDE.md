# StudyFlow Project Guide

## Overview
A student productivity suite using React (Vite), Express, and Supabase.

## Tech Stack
- **Frontend:** React 18, TailwindCSS, GSAP, Recharts.
- **Backend:** Node/Express (Server-side logic & security).
- **Database:** Supabase (Auth, PostgreSQL, RLS).

## Project Structure
- `frontend/src/pages/app/`: Core application views.
- `frontend/src/hooks/`: Data fetching logic (Supabase).
- `backend/server.js`: Express entry point.
- `supabase/schema.sql`: Database tables & RLS rules.

## Commands
- Build: `npm run build`
- Dev: `npm run dev`
